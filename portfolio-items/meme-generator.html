Meme Generator

make yourself laugh

Who said all of these coding projects had to be serious? To get my hands dirty with react, I wanted to make this little web app where you can generate your own memes! I use the ImgFlip API to grab some meme images, and let you randomly select one from the top 100 memes.

Repo Demo

There really isn't a whole lot going on with this app. Mainly, I am using React, along with state management, to be able to allow the user to type in a message that can correspond to the random meme you pull.

I am not actually calling the API directly in this project. In an effort to reduce calls to an API, I had pulled the top 100 meme images, and pulled them into a separate JavaScript file and call them from there. This way, I don't overload an API with too many requests, and let my little web app be as fast as possible.

I am using a form for the inputs, and as it turns out, forms are not as straight forward as they normally are when working with React. Currently, the way I have this working is that every time you type in the form field, the state gets updated.

Every time the "get a new meme image" button is clicked, a random number will be generated between 0 and the lenght of the object holding the memes. Then it will put that random number index meme on the screen for you!

There's a few things I would like to point out with this project:

  • This is not a responsive project. My intent was to focus on the functionality. So responsiveness was not something I was focusing on.
  • Some of the meme images are not going to be in a correct format for this generator. I did not hand-select the meme images that are getting used. So you may see something where the "top text / bottom text" meme format doesn't make sense.
  • This is a minor issue, but in the gif at the top of the page, the cursor appears to not be clicking in the right spot. That was an issue with the recording software, and not my app.
