Travel Journal

my adventures

This was a pretty simple project using React. All this is, is a little page to show some of the recent places I have traveled to. There's nothing too fancy going on here, and it's just some cards that show some places I have been. I included a little thumbnail image, the date, the location, the event, and even a link to Google Maps.

Repo Demo

Under the hood, there isn't a whole lot going on with this one. I have the data for my travels within a separate JavaScript file. Within the code, I have a .map() function that cycles through the object in that JS file. This way, any time I want to add a new destination, I just have to add a new item to my object, and it will automatically create a new entry!

This was more or less just a fun creative project for me. I don't really have any intention on making this a full website, or adding a ton of features to it. This was really just a way for me to get my hands on some keys and code up a little project.
